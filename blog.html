Important FAQ

What is the Document Object Model(DOM)?

DOM means Document object model. It represents the page so that program can modify the document structure, style and content. The DOM represents the document as nodes and objects this is the way programming language can interact with the page.

How do you select an element From the DOM?

There are 5way to select element from the document object model.

  • Using getElementById
  • Using getElementsByClassName
  • Using getElementsByTagName
  • Using querySelector
  • Using querySelectorAll

What is event delegation in the context of the Dom, and why is it useful?

Event delegation uses for event propagation(bubbling) to handle events higher level in the DOM .

How do you manipulate an elements attributes and styles using the DOM?

We can use setAttribute() method to change attributes like id, href, or src . SetAttribute() method accepts two arguments.
